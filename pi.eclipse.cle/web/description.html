Overview

CLE is an Eclipse plugin providing editing of grammar specification files used by CUP and scanner specification files used by JFlex.

The plugin also includes the neccessary builders needed for Java sources generation.

Requirements

  • Eclipse - 3.1.x (developed with 3.1.2)
  • J2SE - 5.0

Installation

CLE can be installing using the Update Manager from Source Force.

See also the downloads area of this project.
